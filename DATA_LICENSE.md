**License**: [Creative Commons Attribution 4.0 International (CC BY 4.0)](https://creativecommons.org/licenses/by/4.0/)

The following fields are **derived works** generated by IMMI-Case's
extraction pipelines (regex + LLM) over the AustLII source texts:

| Field | Method | Coverage (2026-02-22) |
|-------|--------|----------|

## 4. Vector embeddings

Embeddings stored in Supabase PostgreSQL (pgvector HNSW) are derived from
court text content via:

- OpenAI `text-embedding-3-small` (1536-dim)
- Google `gemini-embedding-001`

Both routed via Cloudflare AI Gateway (unified billing).

The embedding **vectors** are derived data and follow §2 (CC BY 4.0).
The choice of embedding model and integration code follow §3 (Apache 2.0).
